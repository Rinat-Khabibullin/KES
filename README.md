# Электрика Туапсе

Одностраничный лендинг для команды электриков в Туапсе и районе.

## Ссылки

- GitHub Pages: https://rinat-khabibullin.github.io/KES/
- Репозиторий: https://github.com/Rinat-Khabibullin/KES

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Деплой на GitHub Pages выполняется автоматически через GitHub Actions при push в ветку `main`.
